~/.zshlocal

Install

Run this single command. It downloads the config, sources it from both ~/.zshrc and ~/.bashrc, and auto-detects your shell.

one-line installer
curl -fsSL https://raw.githubusercontent.com/blaineam/blaineam.github.io/main/dotfiles/zshlocal -o ~/.zshlocal && echo '[ -f ~/.zshlocal ] && source ~/.zshlocal' >> ~/.zshrc && echo '[ -f ~/.zshlocal ] && source ~/.zshlocal' >> ~/.bashrc && source ~/.zshlocal
